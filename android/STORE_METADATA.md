# Google Play — анкета рейтинга контента (IARC / Content Rating Questionnaire)

## Категория приложения
Социальные сети / Мессенджеры

## Ответы на вопросы анкеты IARC

**Насилие:** Нет
**Сексуальный контент:** Нет
**Нецензурная лексика:** Нет
**Вещества (алкоголь, наркотики):** Нет
**Азартные игры:** Нет
**Пользовательский контент (UGC):** Да — пользователи обмениваются текстовыми сообщениями и изображениями
**Возможность связи с другими пользователями:** Да — мессенджер

## Ожидаемый рейтинг
IARC: 3+ (Everyone) — мессенджер без встроенного публичного контента
ESRB: Everyone
PEGI: 3
USK: 0
ClassInd: Livre (Livre)

## Data Safety (раздел в Google Play Console)

| Тип данных         | Собирается | Передаётся | Цель                  | Обязательно |
|--------------------|------------|------------|----------------------|-------------|
| Номер телефона     | Да         | Да         | Функциональность     | Нет         |
| Идентификатор (UIN)| Да         | Да         | Функциональность     | Да          |
| Сообщения          | Нет¹       | Нет¹       | —                    | —           |
| Контакты телефона  | Да²        | Нет²       | Функциональность     | Нет         |
| Токен устройства   | Да         | Да         | Push-уведомления     | Нет         |
| IP-адрес           | Да         | Нет³       | Безопасность         | Да          |

¹ Сообщения шифруются на устройстве (E2E), сервер видит только зашифрованный блоб
² Только HMAC-хеши — сами имена и номера на сервер не уходят
³ IP логируется только для rate-limiting, не передаётся третьим лицам

**Шифрование при передаче:** Да (TLS 1.3 + транспортное ChaCha20)
**Возможность запроса удаления данных:** Да (через настройки приложения)

# RuStore — чеклист публикации

## Обязательные поля
- Название: АйРомашка — мессенджер
- Краткое описание: Приватный мессенджер со сквозным шифрованием (≤80 символов)
- Полное описание: см. fastlane/metadata/android/ru-RU/full_description.txt
- Категория: Социальные сети
- Возрастной рейтинг: 3+
- Email разработчика: support@iromashka.ru
- Политика конфиденциальности: https://iromashka.ru/privacy-policy.html
- Пользовательское соглашение: https://iromashka.ru/terms.html

## Требования к APK / AAB для RuStore
- minSdk ≥ 21 ✓ (у нас 26)
- targetSdk ≥ 31 ✓ (у нас 34)
- Подпись: release keystore (не debug) ✓
- Package name: com.iromashka

## Ограничения RuStore
- Нет требований по GooglePlay Services — приложение их не использует ✓
- Firebase/FCM: если используется, нужно указать в анкете (у нас опционально, через настройки)

## Скриншоты (нужно подготовить вручную)
- Минимум 4 скриншота телефона (1080×1920 или 1080×2340, PNG/JPG)
- Feature Graphic: 1024×500 px
- Иконка: 512×512 px (уже есть: ic_launcher-playstore.png)

Рекомендуемые скриншоты:
1. Экран входа / регистрации
2. Список чатов / контактов
3. Открытый чат (переписка)
4. Настройки профиля / статус
