АйРомашка — российский мессенджер с открытым исходным кодом, построенный на принципах приватности и безопасности.

🔒 Сквозное шифрование (E2E)
Все сообщения шифруются на вашем устройстве алгоритмом P-256 ECDH + AES-256-GCM. Сервер не имеет доступа к содержимому переписки.

📱 Мультиустройство
Работает одновременно на нескольких устройствах. История синхронизируется в зашифрованном виде.

👥 Групповые чаты
Поддержка групповых переписок с полным сквозным шифрованием для каждого участника.

🔑 Резервное восстановление
12-словная BIP39-фраза восстановления позволяет вернуть доступ к истории чатов даже при утере устройства.

📖 Синхронизация контактов
Автоматический поиск знакомых по телефонной книге — без передачи самих номеров на сервер (HMAC-хеширование).

🛡 Приватный ключ — только у вас
Приватный ключ шифруется на устройстве и никогда не покидает его.

⚡ Быстрый вход (Fast-unlock)
Разблокировка по биометрии или PIN-коду без повторного ввода пароля.

📦 Открытый исходный код
Код приложения доступен для аудита на GitHub.

Требования: Android 8.0 и выше.
Сервер расположен в России. Данные обрабатываются в соответствии с ФЗ-152.
